# 05. UI仕様書：グルーヴ編集

## 1. 概要 (Overview)
「グルーヴ感の重要性」を実現するため、トラック毎・小節毎にグルーヴの「種類」と「度合い」を編集する機能。

## 2. トラック毎のグルーヴ (曲全体のノリ)

* **UI:** 各トラック（`Drums`, `Bass`...）のコントロールパネル（「アレンジ・ビュー」内）に `[Groove]` ボタンを設置する。
* **ワークフロー:**
    1.  ユーザーが `[Bass]` トラックの `[Groove]` ボタンをタップする。
    2.  「グルーヴ・テンプレート」のプリセット・リストが表示される。
    3.  ユーザーが `[ Shuffle (8th) ]` を選択する。
* **結果:** `[Bass]` トラックの**全てのノート**が、自動的に「シャッフル（跳ねた）」リズムのタイミングに**一括で再配置**される。

## 3. 小節毎のグルーヴ (フィルや展開)

* **UI:** `docs/02` で定義した `[ MUTATE (介入) ]` ボタンのワークフローに統合する。
* **ワークフロー:**
    1.  ユーザーが、グルーヴを変えたい**「小節」**（例：4小節目だけ）を**タップして選択**する。
    2.  `[ MUTATE (介入) ]` ボタンを押す。
    3.  ポップアップに「グルーヴの介入」が提案される。
        * **▼ グルーヴの介入**
        * `[ 提案A (J Dilla) ]`: この小節だけ「ヨレた」グルーヴに変更
        * `[ 提案B (Nudge: Push) ]`: この小節だけ「突っ込んだ」タイミングに変更
    4.  ユーザーが選択すると、その小節のグルーヴだけが変更される。

## 4. 度合い (Intensity) の編集
* 上記のテンプレート（例：`[ J Dilla (Lazy) ]`）を適用する際、その「ズラし具合」や「ヨレ具合」の**度合い（Intensity: 0%〜100%）**も、スライダーなどで調整できるようにする。
